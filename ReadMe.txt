For start, you need do next command to your terminal:
1. npm install expess;
2. npm install mongojs;
3. npm install body-parser;
After install this packages, you should start node server with help:
node server (in git bash terminal)
Control database with mongo, start with help:
mongod (in git bash terminal too)
mongo (in git bash terminal too)

AND all is ready for testing :)
